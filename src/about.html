About my Application

Welcome to my web application for managing work experiences! This application allows users to store, retrieve, update, and delete their work experiences conveniently.

Purpose

The primary purpose of this application is to provide users with a platform to organize and track their work history. Whether you're a professional looking to keep a record of your career journey or a student preparing your CV, our application offers a user-friendly interface to manage your work experiences effectively.

Technologies Used

    My web application is built using the following technologies:
  • Node.js: Backend server environment for running JavaScript code.
  • Express.js: Web application framework for Node.js, used for routing and handling HTTP requests.
  • SQLite: Lightweight relational database management system used for data storage.
  • HTML/CSS/JavaScript: Frontend technologies for building the user interface and interactivity.
  • Fetch API: Client-side JavaScript interface for making HTTP requests to the backend server.

    • How It Works

  • Adding Work Experiences: Users can add new work experiences by filling out a simple form in the "Add Work Experience" page. The submitted data is then stored in our database for future reference.
  • Viewing Work Experiences: The "Home" page displays a list of all stored work experiences, allowing users to easily view their career history at a glance.
  • Updating and Deleting: Users have the option to update or delete existing work experiences. This functionality provides flexibility for users to manage their records as needed.

    • About the Database

    Our application utilizes a SQLite database to store work experience data. SQLite is a reliable and efficient choice for small to medium-sized databases, offering robust relational database features without the need for a separate server process.
